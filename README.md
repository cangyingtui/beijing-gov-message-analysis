# 北京市政务留言板数据分析

## 项目简介

本项目基于人民网 **领导留言板北京市版块** 的市民留言及政府回复数据，  
通过数据科学方法对市民诉求与政府响应情况进行系统分析。

研究目标包括：

- 识别市民关注问题的主要类型
- 分析政府部门回复效率
- 探索不同区域之间的差异
- 研究节假日对留言与处理效率的影响

数据时间范围：2024-01-01 至 2025-12-09。

数据来源：人民网领导留言板北京市板块。

## 数据来源

原始数据来自人民网领导留言板网站爬取的数据。  
随后对原始数据进行了结构化整理，并转换为 CSV 数据集。

主要字段包括：

- 留言内容
- 留言时间
- 回复时间
- 承办单位
- 所属行政区
- 办理状态

在数据预处理阶段进一步构建了多个衍生变量，例如：

- 回复耗时
- 是否节假日
- 时间统计字段
- 行政区映射字段

## 项目亮点

### 节假日影响分析

本项目将中国法定节假日信息引入数据分析流程，研究节假日对政务服务运行的影响，包括：

- 节假日期间留言数量变化
- 节假日对政府回复效率的影响
- 节后处理积压情况

该部分分析能够更好地反映公共服务系统的运行压力。

### 网页可视化展示

项目开发了一个网页形式的数据可视化界面，用于展示分析结果。

主要功能包括：

- 时间趋势图
- 各行政区空间分布地图
- 关键词分析面板

通过交互式界面可以更直观地探索市民诉求的变化趋势。

## 方法流程

本项目的数据分析流程主要包括以下几个部分：

### 1 数据预处理

- 数据清洗
- 字段标准化
- 回复时长计算
- 行政区名称统一

### 2 文本分析

对留言内容进行自然语言处理，包括：

- 中文分词（Jieba）
- 关键词统计
- 词云生成
- 关键词共现网络分析

### 3 时间趋势分析

对留言数据进行时间序列分析，包括：

- 月度留言变化趋势
- 政府回复效率变化
- 节假日影响分析

### 4 空间分析

结合北京市行政区数据进行空间统计：

- 各区留言数量分布
- 各区回复效率对比
- 留言密度空间可视化

### 5 数据可视化

项目生成多种可视化结果，包括：

- 高频词统计表
- 词云图
- 关键词共现网络
- 时间趋势图
- 空间分布地图
